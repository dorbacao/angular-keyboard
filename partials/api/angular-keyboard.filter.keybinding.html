keybinding
filter in module angular-keyboard

Description

Formats a keybinding string into a user accessible string.

Usage

In HTML Template Binding

{{ keybinding_expression | keybinding[:useHTML] }}

In JavaScript

$filter('keybinding')(input[, useHTML])

Parameters

ParamTypeDetails
inputstring

The string representing the keybinding.

useHTML
(optional)
boolean

Set this flag to true if you are using ng-bind-html to bind the output, as it will output proper <kbd> tags.

Returns

string

A UTF-8 string with special characters replaced and character sequences represented by arrows or a $sce trusted string representing the same in acii HTML.
