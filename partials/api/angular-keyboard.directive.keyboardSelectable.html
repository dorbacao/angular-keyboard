keyboardSelectable
directive in module angular-keyboard

Description

Used for performing keyboard actions on lists of items. It must be specified on the parent element as ng-repeat and exposes another scope variable to it's elements: keyboardSelected which is a boolean indicating selection on the particular class.

To trigger keyboard shortcuts only on the selected element, use keyboardShortcut's selection-shortcut option to make it work only on the selected element.

Usage

as attribute
<ANY keyboard-selectable="{string}"
     [selected-class="{string}"]
     [select-next="{string}"]
     [select-previous="{string}"]
     [keyboard-prevent-default="{boolean}"]
     [private-shortcut="{boolean}"]
     [selection-action="{string|Function(element, collection) }"]>
   ...
</ANY>

Parameters

ParamTypeDetails
keyboardSelectable
(optional)
string

The name of the kind of thing being selected. This, unless privateShortcut is set will cause two entries to be added to the keyboard shorcuts list: 'Select next {{keyboardSelectable}}' and 'Select previous {{keyboardSelectable}}'.

selectedClass
(optional)
string

The css class used to establish selection. Defaults to keyboard-selected.

selectNext
(optional)
string

The keybinding string that will select the next element in the collection.

selectPrevious
(optional)
string

The keybinding string that will select the previous element in the collection.

keyboardPreventDefault
(optional)
boolean

Prevents the default keyboard action.

privateShortcut
(optional)
boolean

Hides the shortcut from the keyboardHelp directive and from exposure in KeyboardShortcuts.actions().

selectionAction
(optional)
stringFunction(element, collection)

This attribute decides what to do on selection. The default value, 'class' will remove the selectedClass from all other elements in the collection and add it to the selected element. Another string value 'click()' will send a click event on selection to the element. You can specify a css selector between the brackets and the click event will be sent to any children of the element matching the selector (e.g. 'click(a.link)'). Finally you may provide a custom function with this functionality.
