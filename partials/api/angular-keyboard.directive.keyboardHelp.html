keyboardHelp
directive in module angular-keyboard

Description

Displays a formatted table of the currently active keyboard shortcuts. The directives that define these shortcuts need to be loaded in the current view.

The HTML that this directive generates (for styling purposes) looks like this: 

<table>
 <tr>
   <td><kbd>⌥</kbd> <kbd>S</kbd></td>
   <td>Save the document</td>
 </tr>
 <tr>
   <td><kbd>g</kbd> <span class="keyboard-separator">›</span> <kbd>i</kbd></td>
   <td>Go to issues</td>
 </tr>
</table>

The keybinding are formatted via the keybinding filter.

Usage

as element:
<keyboard-help>
</keyboard-help>
as attribute
<ANY keyboard-help>
   ...
</ANY>

Directive info

  • This directive creates new scope.
